##  スワイプの仕組みは？

<br>

1. 移動する方向が先頭になるように並び替える
  - <y>[2, 0, 4, 4] -> [4, 4, 0, 2]</y> （右移動の場合）
2. 0以外の数字を先頭に移動させる
  - <y>[4, 4, 0, 2] -> [4, 4, 2, 0]</y>
3. 隣合う数字が同値であれば、合体させる
  - <y>[4, 4, 2, 0] -> [8, 0, 2, 0]</y>
4. もう一度0以外の数字を先頭に移動させる
  - <y>[8, 0, 2, 0] -> [8, 2, 0, 0]</y>
